Terminology Decoder

Git Concepts Translated

Hover over cards to see tooltips, or click buttons to run actions. Here is what they actually do:

  1. git add // Prepare edits
    Moves local edits from your Project Files into the Prepared checklist.
  2. git commit // Save snapshot
    Saves the prepared checklist as a permanent snapshot in your Local History.
  3. git push // Upload to cloud
    Copies local snapshots to Cloud Storage (GitHub).
  4. git pull // Download from cloud
    Downloads the latest cloud snapshots to your machine.
  5. git branch // Create draft workspace
    Splits off an isolated timeline to test draft features safely.
  6. git merge // Combine drafts
    Merges your draft workspace timeline back into main.
